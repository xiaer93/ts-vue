Vue.component('button-counter', {
  data: function () {
    return {
      count: 0
    }
  },

  render () {
    return h('button', {
      on: {
        click() {
          this.count += 1
        }
      }
    }, `you click ${this.count}`)
  }
})


组件的props如何传递

创建组件vnode（有属性propData）---实例化组件（有属性propOptions）----验证属性，同时添加响应式

当属性发生变化时，通过render重新计算组件vnode，实现父节点向子节点传递消息
